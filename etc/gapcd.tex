\begin{center}
{\Large Quick Installation}
\medskip\\
\end{center}
Decide whether you want to install {\sf GAP} 3, {\sf GAP} 4 or both.\bigskip\\
\begin{minipage}[t]{53mm}
\begin{center}
{\bf GAP 3}
\end{center}
\begin{raggedright}
{\small
\begin{description}
\item UNIX:
Copy the {\tt gap3r3p4} directory to your local disk. Go to the {\tt src}
subdirectory. Call {\tt make} and then {\tt make} with the correct
architecture.
\item Macintosh:
Install the Macintosh files from the {\tt ftp/gap-3.4.4/bin/mac} directories.
\item Windows:
Go to the {\tt gap3r4p4} directory. Start with
\begin{center}
{\tt bin$\backslash$gapdjg.exe -l lib}
\end{center}
\end{description}
For further details see the {\tt README} file in the {\tt ftp/gap-3.4.4}
directory. If problems occur, install from scratch from this {\tt ftp}
directory. 
}
\end{raggedright}
\end{minipage}\quad%
\begin{minipage}[t]{53mm}
\begin{center}
{\bf GAP 4}
\end{center}
\begin{raggedright}
{\small
\begin{description}
\item UNIX:
Copy the {\tt gap4b4} directory to your local disk. Go into this directory.
Call {\tt ./configure} and {\tt make}.
\item Macintosh:
Install the Macintosh files from the {\tt ftp/gap4/bin/mac} directories.
\item Windows95:
Go to the {\tt gap4b4} directory. Start with
\begin{center}
{\tt bin$\backslash$gapw95.exe}
\end{center}
\end{description}
For further details see the {\tt INSTALL} files in the {\tt ftp/gap4}
directory. If problems occur, install from scratch from this {\tt ftp}
directory. 
}
\end{raggedright}
\end{minipage}\bigskip\\
After you installed {\sf GAP}, please send us an email at {\tt
gap@dcs.st-and.ac.uk}.
